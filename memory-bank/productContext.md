# Product Context

## 为什么存在
用户在 QQ 群聊中经常分享各平台的链接，但这些链接打开不便（需要跳转浏览器/APP）。本插件自动解析链接内容并以图文卡片形式直接展示，提升群聊体验。

## 解决的问题
- 群聊中的链接无法直接预览内容
- 不同平台的分享链接格式各异，需要统一处理
- 视频/图集等媒体内容需要下载后才能在群聊中分享

## 用户体验目标
1. **零操作** - 发送链接即自动解析，无需手动触发
2. **快速响应** - 异步下载 + 缓存机制保证响应速度
3. **美观展示** - PIL 渲染的卡片包含封面、标题、作者、数据等信息
4. **媒体转发** - 视频和图集自动下载并发送到群聊
5. **可配置** - 管理员可按群开关，支持多种渲染模式
